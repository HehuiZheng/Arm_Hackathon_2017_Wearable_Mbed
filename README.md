# Arm_Hackathon_Wearable_Mbed
Wearable device built by MBed helps people keep track of their time allocation in a day, and keep an eye on their surroundings.

Major Activities Monitored: 
1. Browsing the Internet and writing
2. Walking
3. Idle

Special features of our device
1. Mbed directly upload data to the server through MQTT
2. Automatically recignize movement patterns of user with neural network
3. Sending warning message to user’s mailbox when abnormality is detected in environment (Air quality and Ambient light)
4. Raise to wake: show all the readings including time on Mbed’s display only when user raise the device towards his/her face.
